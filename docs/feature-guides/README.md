# Potraces — Feature Guides (Word)

Plain-language, **code-accurate** Word documents — one per feature — so anyone can see clearly
what each feature does, every way to use it, and what's manual / not possible.

Each `.docx` is generated by its `gen_*.py` script using the shared template `_docgen.py`
(on-brand earthy palette, colour-coded tables, no harsh red). To edit a doc, edit its script and
re-run it — never hand-edit the `.docx`.

## How to regenerate

```powershell
cd docs/feature-guides
python gen_add_expense.py   # → 01-Adding-Money-Every-Method.docx
python gen_wallet.py        # → 02-Wallets.docx
python gen_debt.py          # → 03-Debts-Splits-Shared-Subscriptions.docx
```

(Requires `python-docx`: `pip install python-docx`.)
