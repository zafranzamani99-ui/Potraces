# Instructions to Run the App

### **1. Clone the repository**
```bash
git clone https://github.com/zafranzamani99-ui/Potraces.git
```

### **2. Navigate to the project directory**
```bash
cd Potraces
```

### **3. Install dependencies**
```bash
npm install
```

### **4. Install the required SDKs**
- If your app uses **Expo** or **React Native**, ensure the following:
  - Install the **Expo CLI** globally:
    ```bash
    npm install -g expo-cli
    ```
  - Install the **Android Studio** (for Android emulator) or **Xcode** (for iOS simulator) if running on a simulator.
  - Ensure the **Android SDK** is installed and properly configured in Android Studio.

### **5. Start the app**
- For Expo:
  ```bash
  expo start
  ```
- For React Native (if not using Expo):
  ```bash
  npx react-native start
  ```

### **6. Run the app on a device or emulator**
- For Android:
  ```bash
  npx react-native run-android
  ```
- For iOS (macOS required):
  ```bash
  npx react-native run-ios
  ```

---

### **Prerequisites**
- **Node.js**: Install from [Node.js official website](https://nodejs.org/).
- **npm**: Comes with Node.js.
- **Android Studio**: Download from [Android Studio](https://developer.android.com/studio).
- **Xcode** (for iOS): Install from the Mac App Store.

---

These steps ensure that you have all the necessary tools and dependencies to run the app successfully.
