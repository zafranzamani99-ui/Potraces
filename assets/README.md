# Assets Directory

This directory should contain the app's visual assets referenced in `app.json`.

## Required Files

### App Icons

- **icon.png** (1024x1024px)
  - Main app icon
  - Should be square with no transparency
  - Represents FinFlow brand

- **adaptive-icon.png** (1024x1024px, Android only)
  - Android adaptive icon foreground
  - Should work on various background colors

- **splash.png** (1284x2778px for iPhone 12/13 Pro Max)
  - Launch screen image
  - Background color set in app.json: #6366F1 (primary blue)

- **favicon.png** (48x48px, Web only)
  - Browser tab icon for web version

## Design Guidelines

### Color Scheme
- Primary: #6366F1 (Indigo)
- Secondary: #10B981 (Green)
- Use these colors for consistency

### Icon Concepts
Consider incorporating:
- Dollar sign or currency symbol
- Upward trending arrow (growth/finance)
- Dual circles or split design (representing Personal/Business modes)
- Clean, modern, minimal design

## Temporary Solution

For development, you can use placeholder images or generate icons using:
- [Icon Kitchen](https://icon.kitchen/)
- [App Icon Generator](https://www.appicon.co/)
- [Figma](https://www.figma.com/) for custom design
